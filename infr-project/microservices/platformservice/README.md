# Platform Service

## 1. Overview

This project, part of **`Projectarium`**, is used to manage third-party platforms.

## 2. Description

Currently supports three platforms: github, circleci, and vercel..

## 3. Deployment

- **Vercel**: When deployed on the `Vercel`, this service and other `Projectarium` services will be deployed as a unified whole.

- **other**: Can be deployed independently
